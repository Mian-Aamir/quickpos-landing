## 🚀 How to Run

### Requirements
- MAMP installed on your computer

### Steps
1. Clone the repository:
   git clone https://github.com/Mian-Aamir/quickpos-landing.git

2. Move folder to MAMP htdocs:
   C:/MAMP/htdocs/quickpos-landing

3. Start MAMP and turn on Apache server

4. Open browser and go to:
   http://localhost/quickpos-landing/index.php
